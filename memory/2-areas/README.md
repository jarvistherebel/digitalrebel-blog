# Areas of Responsibility

Areas = ongoing standards (no deadline). Review weekly.

## Standard Areas

- **Health** — fitness, sleep, nutrition, mental health
- **Finance** — budget, investments, taxes, spending
- **Career** — skills, network, reputation, growth
- **Relationships** — family, friends, community
- **Learning** — books, courses, experiments
- **Environment** — home, workspace, tools

## Area Template

Each area folder contains:
- `README.md` — what's the standard? what's the current state?
- `checklist.md` — recurring tasks and review items
- `resources/` — related notes, articles, tools
