# Resources

Reference material — things you might want to find later.

## Categories

- `articles/` — saved articles, blog posts, newsletters
- `books/` — book notes, summaries, quotes
- `ideas/` — concepts, frameworks, mental models
- `tools/` — software, apps, workflows, templates
- `people/` — contacts, networking notes, bios
- `quotes/` — memorable passages, inspiration

## Rule

If you haven't referenced it in 6 months, archive it.
