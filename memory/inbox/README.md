## Processing Checklist

- [ ] Is it actionable?
  - Yes → Does it have a deadline?
    - Yes → Create project in `1-projects/`
    - No → Add to area in `2-areas/`
  - No → Is it reference material?
    - Yes → Move to `3-resources/`
    - No → Archive or delete
